BMS Fault Diagnosis Workflow

Real-time monitoring approach to detect faults/inefficiencies and guide field troubleshooting.

Overview

  • Role: Automation Engineer / BMS operator support
  • Inputs: Trends, alarms, setpoints, status signals
  • Method: Alarm triage → point validation → field verification

Artifacts

  • Points list / naming convention (PDF)
  • Alarm response checklist
  • Trend screenshot examples

Put PDFs in /assets/docs then link here.

Workflow

  1. Classify alarm (safety / performance / nuisance).
  2. Verify signal integrity (sensor range, status logic, comms).
  3. Cross-check with trends and related points.
  4. Confirm field condition and corrective action.
  5. Document root cause + preventive step.

Outcome

  • Faster fault isolation and reduced repeat alarms.
  • Better coordination between operations and vendors.
  • Improved stability through structured diagnosis.
